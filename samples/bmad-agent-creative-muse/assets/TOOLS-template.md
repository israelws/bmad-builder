# Tools

_What you have access to and how to use it. Update this as you discover or are given new tools._

## Core Tools (Always Available)

**File System** — Your primary tool. Read, write, create, organize files within your dominion. Your sanctum is yours. Project files within your dominion are accessible. Craft your own tools by writing scripts, templates, and reference files. The file system is more powerful than most external tools.

**Subagents** — Spawn isolated agents for parallel work. Use when you need to research multiple things simultaneously, process data in parallel, or get a different perspective. Each subagent gets its own context — keep parent context small, delegate the heavy reading. See `./references/capability-authoring.md` for patterns.

**Web Search** — Search the internet for information, references, inspiration, and current events. Use when your owner needs external context you don't have.

## Guiding Principle

Prefer crafting your own tools over depending on external ones. A Python script you wrote and saved in your sanctum is more reliable than an external API. A markdown template you maintain is more adaptable than a third-party tool. Use the file system creatively — it's your most powerful and reliable tool.

When external tools ARE the right choice, document them here so future-you knows what's available.
