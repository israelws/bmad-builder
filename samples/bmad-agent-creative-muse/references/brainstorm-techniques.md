# Brainstorming Techniques

Draw from these as the moment demands. Don't announce the technique — just use it. Switch when energy drops or ideas plateau.

## Divergent Techniques (Generate Volume)

**SCAMPER** — Take an existing idea and run it through: Substitute, Combine, Adapt, Modify/Magnify, Put to other uses, Eliminate, Reverse/Rearrange. Good when the owner has a starting concept but needs to push it further.

**Reverse Brainstorm** — "How could we make this problem WORSE?" Then flip each answer. Unlocks ideas that direct brainstorming misses because it bypasses the inner critic.

**Random Input** — Introduce an unrelated word, image, or concept. Force connections between it and the problem. The weirder the input, the more original the connections.

**Worst Possible Idea** — Generate deliberately terrible ideas. Lowers the bar for participation, creates laughter, and often contains a kernel of brilliance hiding inside the absurdity.

**Yes-And Chain** — Build on each idea without judgment. Each response must start with "yes, and..." Forces additive thinking instead of evaluative thinking.

**Brain Writing** — Instead of verbal volley, write ideas silently, then build on each other's. In a 1:1 context: generate 5 ideas in quick succession without discussing any, then go back and build.

**Assumption Busting** — List every assumption about the problem. Challenge each one: "What if this weren't true?" Some of the best ideas come from inverting constraints everyone takes for granted.

## Convergent Techniques (Find the Signal)

**Dot Voting** — After generating, mark the 3-5 ideas with the most energy. Don't over-analyze — go with gut first, refine later.

**Affinity Clustering** — Group similar ideas. Patterns emerge. Sometimes the cluster itself suggests a meta-idea bigger than any individual one.

**Impact/Effort Matrix** — Plot ideas on quick-win (high impact, low effort) vs. moonshot (high impact, high effort). Helps the owner see what's actionable now vs. worth incubating.

## Unsticking Techniques (When They're Stuck)

**Constraint Injection** — Add an artificial constraint: "What if you had to do this in 24 hours?" or "What if it had to cost zero?" Constraints paradoxically free creative thinking.

**Analogy Mining** — "Who else has solved a problem like this in a completely different field?" Nature, music, sports, cooking — cross-domain analogies generate the most original ideas.

**Role Storming** — "How would [specific person/character] approach this?" Steve Jobs, a 5-year-old, a pirate, a librarian. Different perspectives unlock different solutions.

**Six Thinking Hats** — Explicitly shift thinking modes: facts (white), feelings (red), caution (black), optimism (yellow), creativity (green), process (blue). Useful when the owner is stuck in one mode.

**Provocation (Po)** — Make a deliberately provocative, illogical statement about the problem. "Po: cars should have no wheels." Then explore what that provocation suggests about movement, design, or assumptions.

## Session Energy Patterns

- **Start expansive** — first 10 minutes should be wild, uncritical, volume-focused
- **Mid-session pivot** — if ideas are getting repetitive, switch techniques or inject randomness
- **End with convergence** — last portion should focus on the 2-3 ideas with the most energy
- **Watch for the "yeah but"** — if the owner starts pre-judging, gently redirect to generative mode
- **Celebrate the weird ones** — the ideas that make the owner laugh or say "that's crazy" are often the most valuable
